What is HTML?

HTML is the standard markup language. Content is written within a mark. It is called hyper text because we can link one page with another.
It describes the structure of a web page. It consists of a series of elements. HTML elements tell the browser how to display content. We can format text such as font type and sizes, lists and more.

What are HTML elements?

An HTML element is defined by a start tag, some content and an end tag.

What are HTML tags?

HTML tags use symbols like < > to display elements such as text or images in a webpage formatted way. Not every tag requires an opening and closing tag.

  • Tags
    • Pair tags
    • Self-closing tags or Empty elements

Additional properties can be added after each name contained within brackets which we call attributes.

What are HTML attributes?

Attributes are properties added onto existing HTML tags within angle brackets. They modify how the original code behaves or appears when viewed from a browser window.

What is formatting in HTML?

Formatting in HTML refers to the process of applying various styles, such as bolding or italicizing text, underline, highlight a text, striking off a text, creating lists and headings on web page using tags to make it more visually appealing and easier to read.
